Инструкция по запуску:
1) Перед началом работы нужно выдать run.sh права на исполнение при помощи команды chmod +x run.sh
2) Команды для собрки Docker образов: ./run.sh build_generator; ./run.sh build_reporter
3) Запуск генератора данных: ./run.sh run_generator
4) Запуск аналитика данных: ./run.sh run_reporter
5) Вывод структуры проекта: ./run.sh structure
6) Очистить папку Data: ./run.sh clear_data
7) Проверить содежимое папки Data из контейнеров: ./run.sh inside_generator; ./run.sh inside_reporter

Запуск веб сервера:
После создания отчета и заполнения папки Data при помощи команды report_server можно вывести отчёт на локальную страницу. При реализации этого задания я пользовался методами семинара, использовал -d, чтобы можно было выполнять другие команды в терминали во время работы порта, и разобранный веь-сервер nginx. чтобы увидеть отчёт на самой странице после запуска нужно добавить в конец адресной строки /report.html. Чтобы завершить работу нужно ввести в терминал docker stop hw3-web-server. При помощи 73 строки в run.sh я подставил в nginx папку с нашей датой, изменив его путь поиска файлов
